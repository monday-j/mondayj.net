# About this site?

Right, that's what the site is for, but what is this site itself?

It's a Hugo instance running in a Cloudflare Worker instance, which itself pulls from a GitHub repo.

After the initial setup, I wanted this to be as simple to keep running as possible. None of it actually runs on my own machine, which if you told me this 2 years ago, I would have bowed my head in shame. In truth, I rather enjoy not needing to think too hard about "high availability" and "system resources" and "hardened endpoints". I certainly don't want to have to bother with "maintenance". I just want to write something and have it look decently neat on a website that uses my own domain.

I keep the files mirrored on my own hardware since I'm not *completely* ready to give up that level of digital sovereignty, but I'm looking forward to keeping this place simple and easy to write in.

That's enough from me! I spent the last hour or so learning how to spin this up, so I'm going to make the most of it and actually write something now. Seeya!
